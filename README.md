# Banking System - Java

A simple console-based banking system built using Java.  
This project is part of a structured learning challenge to improve OOP concepts and Git/GitHub workflow skills.

---

## Features

- Create bank account
- Deposit money
- Withdraw money
- Balance validation
- Console-based interaction

---

## Project Structure

src/
 ├── model/
 │     └── Account.java
 └── Main.java

---

## Concepts Used

- Object-Oriented Programming (OOP)
- Encapsulation
- Constructors
- Input validation
- Git & GitHub workflow
